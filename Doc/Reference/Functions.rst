**Functions**
==============

A comprehensive list of all functions in the game.
Functions are special string values that can be provided to certain keys to make the game initiate certain actions and behavior.

Use `ctrl + f` via your browser or the search bar on the left to quickly find functions you're looking for.
You can also utilize your text editor's ability to search across multiple files to find examples for each function listed.

**General**
------------

Functions that can be used anywhere functions are stated to work.
